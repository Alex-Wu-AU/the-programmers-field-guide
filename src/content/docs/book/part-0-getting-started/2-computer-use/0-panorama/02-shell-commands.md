Each shell has its own set of commands that it understands.

## Example

The following are some example shell commands.

```zsh
ls -lha
cd /c/Users/andrew/Documents/Code
mkdir Project1
dotnet new console
dotnet add package splashkit
clang++ test.cpp -o test -l SplashKit
```
